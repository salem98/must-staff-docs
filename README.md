# MUST Mobile Staff Guide

Public documentation site for staff using the MUST Mobile app.

**Live site:** https://salem98.github.io/must-staff-docs/

## Structure

| File | Page |
|:-----|:-----|
| `index.md` | Home / landing |
| `clock-in-out.md` | Clock In / Clock Out guide |
| `apply-leave.md` | Leave request guide |
| `apply-expenses.md` | Expense submission guide |
| `attendance.md` | Attendance records & schedule |

## Editing

1. Edit the `.md` files directly on GitHub (use the pencil icon).
2. Commit to `main`.
3. GitHub Pages auto-builds and deploys in ~30-60 seconds.

## Adding real screenshots

1. Drop PNG/JPG files into `assets/images/`.
2. In the relevant `.md` file, replace `assets/images/placeholder.svg` with the new filename.
3. Update the caption below the image.

## Theme

Built with [Just the Docs](https://just-the-docs.com/) via `remote_theme` in `_config.yml`. No local Jekyll build needed — GitHub Pages handles everything.

## Issues

Report inaccuracies, outdated instructions, or missing topics via GitHub Issues on this repo, or contact HR directly.
